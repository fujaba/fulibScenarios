## Installation

Add the following snippets to your `build.gradle`:

```groovy
repositories {
	// ...
	jcenter()
	// ...
}

// ...

dependencies {
	// ...
	// https://mvnrepository.com/artifact/org.fulib/fulibScenarios
	compile group: 'org.fulib', name: 'fulibScenarios', version: '0.4.0'
	// ...
}
```
